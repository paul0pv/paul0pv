### üì° Remote Sensing & Soil Moisture Inversion (IEM-C)
Developing hybrid Physics+ML models for soil moisture retrieval from C-band SAR (Sentinel-1):

- Physical modeling: IEM-B implementation with Hallikainen dielectric and Baghdadi roughness calibration
- Hybrid inversion: Physics-guided neural networks with forward model consistency loss
- Uncertainty quantification: Analytical propagation and Monte Carlo methods with quality mapping
- Validation pipeline: Comprehensive tests against Fung (1992) and Baghdadi (2011) references
- Tech stack: Python, PyTorch, NumPy, physical simulations
